# Set-CFCurrentZone

## SYNOPSIS
Sets the currently targeted zone for other functions

## SYNTAX

```
Set-CFCurrentZone [[-Zone] <String>]
```

## DESCRIPTION
Sets the currently targeted zone for other functions

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Set-CFCurrentZone -Zone 'contoso.com'
```
